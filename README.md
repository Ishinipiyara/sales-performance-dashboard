# Sales Performance Analysis Dashboard

## Overview
End-to-end sales analysis using Python, Excel, and Power BI. 
The dashboard shows KPIs, sales trends, regional and category performance.

## Tools Used
- Python (pandas)
- Excel
- Power BI

## Project Structure
- `data_cleaning.py` → Python script for cleaning and feature engineering
- `superstore_cleaned.xlsx` → Cleaned dataset
- `screenshots/` → Dashboard images

## Features
- Total Sales, Total Profit, Profit Margin KPIs
- Monthly Sales Trend
- Regional & Category Analysis
- Interactive slicers in Power BI

## How to Refresh
1. Run `data_cleaning.py` to update the Excel file
2. Refresh the Power BI dashboard
